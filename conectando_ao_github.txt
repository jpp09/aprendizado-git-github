Primeiro precisamos criar um repositório no github, esse mesmo irá nos mostrar qual é a URL que iremos precisar para amndar os arquivos para o github por https, após isso entraremos no cmd, abriremos o file que desejamos subir para o github, iremos primeiro adicionar o servidor remoto, utilizanddo o git remote add origin('URL https do repositório'), após isso precisamos realizar o primeiro push, que pode ser realizado da seguinte maneira, git push origin master(Caso seja na branch principal). Após isso ele irá pedir  seu user do github(usuário/ ou email), posteriormente irá pedir o password, não sei se são todos mas o meu eu consegui upar somente com a criação de um token, ai no password eu colocava o token, e ele conseguia mandar meus arquivos para o github
